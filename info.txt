# Set up

Create React App
npm create-react-app my-app

Update npm
npm update npm -g

Update nodejs
Install nvm

Install Tailwind
npm i -D tailwindcss postcss autoprefixer
-D means development dependencies

npx tailwindcss init -p
It creates tailwindcss and postcss config files
Edit tailwindcss config file

Add Directives
@tailwind base;
@tailwind components;
@tailwind utilities;
